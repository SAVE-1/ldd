# Useful commands

## Ubuntu
```
# add kernel module
sudo insmod ldd.ko
sudo insmod <file>.ko

# remove a kernel module
sudo rmmod ldd
sudo rmmod <name>

# view kernel log
sudo dmesg

# view return value from last command
echo $?
```
